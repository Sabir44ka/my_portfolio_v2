Обзор проекта

Описание и цели

EVA — это инновационный коммерческий Telegram‑бот, разработанный для автоматизации процесса продаж. Бот использует современные технологии искусственного интеллекта для создания полноценного интернет‑магазина прямо в Telegram: от импорта прайс‑листов и умного поиска до корзины и оплаты.

Цели

01
  • Автоматизация процесса продаж и снижение ручной нагрузки
  • Полноценный магазин внутри Telegram: каталог, корзина, оформление заказа
  • Интеграции с оплатами, CRM и аналитикой

Роль и вклад

02

Архитектура и разработка бота, настройка AI/ML‑интеграций, импорт/обработка данных, построение каталога и корзины, интеграция платежей и CRM, админ‑панель, аналитика, деплой и поддержка.

Технологии

03
Стек:
Python 3.11 Aiogram 3.x PostgreSQL Redis OpenAI API Docker SQLAlchemy Pandas
